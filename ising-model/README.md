# 🔥 Ising Model (Triangular Lattice)

## 🧠 Overview
This project explores the Ising model on a **2D triangular lattice**, focusing on phase transitions and collective behavior of spins.


## ⚙️ Model
- Spins: ±1 on each lattice site  
- Interaction: nearest neighbors (z = 6)  
- Hamiltonian: ferromagnetic coupling  


## 🖥️ Method
Simulation performed using the **Metropolis algorithm**:
- Random spin flips  
- Accepted based on energy change and temperature
